### 🔧 OpenClaw Integration

**How to Enable:**

1. **Pre-load wrapper:**
   ```bash
   cd /home/localadmin/.openclaw/workspace
   python3 skills/openclaw-skill-logger.py
   openclaw gateway start
   ```

2. **Or use in code:**
   ```python
   from skills.openclaw-skill-logger import tracker
   
   # Wrap skill execution
   def execute_skill(skill_name, *args, **kwargs):
       with tracker.track_skill_invocation(skill_name, kwargs.get('task', 'skill')):
           return original_execute(skill_name, *args, **kwargs)
   ```

3. **Or add to openclaw.json:**
   ```json
   {
     "hooks": {
       "internal": {
         "entries": {
           "skill-logger": {
             "enabled": true,
             "script": "skills/openclaw-skill-logger.py"
           }
         }
       }
     }
   }
   ```
