### 🔐 CertForge Docker Architecture (2026-03-13 20:00-20:50 UTC)

**Completed:** Full PKI infrastructure with web management, auth, ACL, and OCSP

**Location:** `workspace/build/certforge/`

**Architecture:**
- Root CA container (offline, no network, manual execution)
- Intermediate CA container (online, Flask API on port 5001)
- Central orchestrator API (Flask on port 5000) - JWT auth + ACL
- OCSP responder (port 8888) - Online certificate status
- React admin portal (web UI on port 3000)

**Files Created:**
- `api/Dockerfile`, `api/app.py` (JWT + ACL), `api/acl.py`, `api/ocsp.py`
- `api/requirements.txt` (added pyjwt)
- `root-ca/Dockerfile`
- `intermediate-ca/Dockerfile`, `backend/app.py`
- `intermediate-ca/config/ocsp-extensions.cnf`
- `docker-compose.yml` (added ocsp-responder service)
- `admin-portal/` - React admin UI
- `README.md` - Updated documentation

**Authentication (JWT):**
- `/api/auth/login` - User login
- `generate_token()` - Create JWT with 24h expiry
- `verify_token()` - Validate tokens
- `@require_auth()` decorator - Protect endpoints
- `@require_permission()` decorator - RBAC (read/write/admin)
- In-memory users: `admin` (all perms), `user` (read/write)

**ACL System:**
- `ACLManager` class - File-based ACL storage (`acl.json`)
- `add_user_acl()` - Grant permissions to users
- `add_certificate_acl()` - Restrict cert access by user/role
- `check_user_permission()` - Verify permissions
- `check_certificate_access()` - Verify cert access
- Blueprint endpoints: CRUD for user/cert ACLs

**OCSP Responder:**
- `OCSPResponder` class - Signs OCSP responses
- `generate_responder_key()` - Creates responder cert/key
- `sign_ocsp_response()` - Signs status responses
- `get_cert_status()` - Checks cert validity/CRL
- HTTP endpoint: `POST /ocsp` (port 8888)
- Returns: good/revoked/unknown status

**Deployment Command:**
```bash
cd workspace/build/certforge
docker compose build
docker compose up -d
```

**API Endpoints:**
- `POST /api/auth/login` - JWT login
- `GET /api/health` - Health check
- `GET /api/containers` - List containers
- `POST /api/containers/:name/{start|stop|restart}` - Container control (write perm)
- `GET /api/certs` - List certificates
- `POST /api/intermediate/issue` - Issue cert (write perm)
- `POST /ocsp` - OCSP responder endpoint
