assistant: Matrix uses several ports depending on the protocol:

## **Standard Matrix Ports**

### **HTTP API (most common)**
- **Port 8008** - Default Synapse HTTP API
- **Port 8448** - Alternative HTTP port
- **Port 80** / **443** - If using web server reverse proxy

### **Federation**
- **Port 8448** - Matrix federation (HTTPS)
- **Port 80** - Fallback for federation

### **WebSocket**
- **Port 8447** - Matrix WebSocket federation
