# PKB — Personal Knowledge Base

> Welcome to your personal knowledge base. This vault is for thinking, connecting ideas, and building mental models.

---

## 📁 Vault Structure

### `inbox/`
Quick capture. Raw thoughts, snippets, anything not yet organized.
**Action:** Process weekly → move to proper homes

### `daily/`
Daily notes (YYYY-MM-DD format)
**Use:** Scratchpad for the day, meeting notes, transient thoughts

### `areas/`
Ongoing responsibilities and active projects
- `/areas/projects/` — Active work with end dates
- `/areas/learning/` — Current learning topics
- `/areas/systems/` — Systems you maintain

### `resources/`
Reference material you return to
- `/resources/guides/` — How-to guides (link to existing Obsidian guides vault)
- `/resources/concepts/` — Mental models, frameworks
- `/resources/people/` — Contacts, collaborators

### `templates/`
Note templates for consistency

### `archives/`
Completed projects, old areas, reference material no longer active

### `graphs/`
Exported graph images, visualizations

## 💡 Usage Philosophy

**This vault is for:**
- Connecting ideas across domains
- Building mental models
- Retrospectives and learning
- "Why" context (playbooks have the "how")

**Not for:**
- Active incident response (use ITIL playbooks)
- Raw session logging (use OpenClaw memory)
- Step-by-step procedures (use ITIL playbooks or Guides vault)
