# SSHFS Installation

To install SSHFS on a Windows OS, follow the steps below:

1. Download the latest version of WinFsp from the official website: [https://github.com/billziss-gh/winfsp/releases/latest](https://github.com/billziss-gh/winfsp/releases/latest). Be sure to select the correct version for your operating system (32-bit or 64-bit).
2. Install WinFsp by double-clicking on the downloaded file and following the prompts.
3. Download the latest version of SSHFS-Win from the official website: [https://github.com/billziss-gh/sshfs-win/releases/latest](https://github.com/billziss-gh/sshfs-win/releases/latest). Again, be sure to select the correct version for your operating system (32-bit or 64-bit).
4. Install SSHFS-Win by double-clicking on the downloaded file and following the prompts.
5. Once the installation is complete, open a new command prompt window by pressing "Windows Key + R" and typing "cmd" in the run dialog box.
6. In the command prompt window, type "sshfs" to ensure that the installation was successful. If successful, you should see a list of available options for SSHFS.

To install SSHFS Win Manager on a Windows OS, follow the steps below:

1. Download the latest version of SSHFS Win Manager from the official website: [https://github.com/rafamadriz/sshfs-win-manager/releases/latest](https://github.com/rafamadriz/sshfs-win-manager/releases/latest).
2. Extract the downloaded ZIP file to any directory of your choice.
3. Double-click on the extracted "sshfs-win-manager.exe" file to launch the program.
4. In the program, click on the "Add" button to add a new SSHFS connection.
5. Enter the required details, such as the remote host name or IP address, the username, and the password.
6. Click on the "Mount" button to mount the SSHFS connection.
7. The mounted SSHFS connection will now appear as a new drive in File Explorer.
