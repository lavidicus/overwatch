# Linux Net Tools

## Introduction

In this document, we will be discussing how to install the `net-tools` package on both Ubuntu and Redhat based Linux systems. The `net-tools` package provides several networking tools such as `ifconfig`, `netstat`, and `route` that are essential for network troubleshooting and monitoring.

## Installing net-tools on Ubuntu

To install the `net-tools` package on Ubuntu, follow these steps:

1. Open a terminal on your Ubuntu system.
2. Type the following command and press Enter:

```
sudo apt-get update

```

This command updates the package list to ensure that you can install the latest version of the `net-tools` package.

1. Type the following command and press Enter:

```
sudo apt-get install net-tools

```

This command installs the `net-tools` package on your Ubuntu system.

## Installing net-tools on Redhat

To install the `net-tools` package on Redhat, follow these steps:

1. Open a terminal on your Redhat system.
2. Type the following command and press Enter:

```
sudo yum update

```

This command updates the package list to ensure that you can install the latest version of the `net-tools` package.

1. Type the following command and press Enter:

```
sudo yum install net-tools

```

This command installs the `net-tools` package on your Redhat system.

## Conclusion

In this document, we have discussed how to install the `net-tools` package on both Ubuntu and Redhat based Linux systems. With the installation of this package, you can now use essential network troubleshooting and monitoring tools such as `ifconfig`, `netstat`, and `route`.
