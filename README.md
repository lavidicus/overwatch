# OpenClaw Workspace

This repository contains the OpenClaw configuration, scripts, and custom skills that power the **OpenClaw** assistant. The workspace is intentionally minimal: all source files are version‑controlled, and any stateful data is stored under `memory/` and `backup/`.

## How to use

* **Git workflow** – All changes are committed and pushed automatically by the cron jobs defined in `crontab`. You can also manually run `git add . && git commit -m "msg" && git push` when needed.
* **Agent configuration** – OpenClaw reads `openclaw.json` for agent settings. Edit it to change models, providers, or default agents.
* **Custom skills** – Add a new skill under `skills/`. Follow the [Skill Creator](https://clawhub.com) guidelines to package it.
* **Memory** – Daily notes are written to `memory/` with the format `YYYY-MM-DD.md`. These are not pushed automatically; commit them manually when you want to preserve a snapshot.
