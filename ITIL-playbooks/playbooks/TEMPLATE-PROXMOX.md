# Proxmox VE - Guide Template

This template provides a standardized format for Proxmox VE operational guides.

## Structure

### 1. Overview
Brief description of what this guide covers.

### 2. Prerequisites
- Hardware requirements
- Software requirements
- Network requirements
- Access requirements

### 3. Installation/Setup Steps
Numbered steps with commands and expected output.

### 4. Verification
Commands to verify successful setup.

### 5. Configuration
Additional configuration steps.

### 6. Troubleshooting
Common issues and solutions.

### 7. References
Links to official documentation.

## Usage

Create a new guide by copying this template and filling in:
- **Guide name** in the title
- **Specific content** for the topic
- **Commands** specific to the setup
- **Verification steps** for the configuration

## Example Topics

- Ceph storage setup
- LVM-Thin configuration
- ZFS pool creation
- Network bridge configuration
- HA cluster setup
- Backup server integration
- SSL certificate installation
- Monitoring setup
- User/role configuration
- Disaster recovery procedures

## Naming Convention

Use format: `PROXMOX-[TOPIC].md`

Example:
- `PROXMOX-CEPH-SETUP.md`
- `PROXMOX-ZFS-CONFIG.md`
- `PROXMOX-HA-CLUSTER.md`
