# Self-Improvement Learning Prompt

Review all notes from the past week and identify:

## Learning Opportunities
1. What broke or went wrong?
2. What could have been prevented?
3. What patterns emerge from failures?
4. What lessons should be formalized into rules?

## Improvement Recommendations
1. Process changes that would prevent recurring issues
2. Tools or automation that could reduce manual work
3. Knowledge gaps that need to be addressed
4. Best practices that should be documented

## Cross-Domain Insights
1. Patterns that appear in multiple domains
2. Principles that could be applied more broadly
3. Opportunities for innovation through connection
4. Potential risks that need monitoring
