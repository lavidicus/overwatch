# Monthly Review Prompt

Review all notes from the past month and generate:

## Top 3 Improvements
1. What was the most impactful learning?
2. What process change would have the biggest effect?
3. What pattern should be monitored going forward?

## Key Metrics
1. Number of notes created by category
2. Number of connections made
3. Number of lessons learned
4. Number of improvements implemented

## Recommendations
1. What should be done differently next month?
2. What processes need refinement?
3. What new areas should be explored?
4. What risks need attention?
