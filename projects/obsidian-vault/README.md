# Obsidian Vault System

## Overview
This vault captures, organizes, and synthesizes our learning, insights, and project work for long-term reference and continuous improvement.

## Structure
```
obsidian-vault/
├── index.md                    # Main index and documentation
├── 01-Daily/                   # Daily notes from memory/ directory
├── 02-Weekly/                  # Weekly summaries and insights
├── 03-Monthly/                 # Monthly reviews and trends
├── 04-Projects/                # Project documentation and progress
├── 05-Learnings/               # Self-improvement and lessons learned
├── 01-Observations/            # Raw observations and data points
├── 02-Reactions/               # Reactions and initial responses
├── 03-Patterns/                # Identified patterns and trends
├── 04-Numbers/                 # Quantitative data and metrics
├── monthly-reviews/            # Historical monthly reviews
└── templates/                  # Note templates
```

## Maintenance Schedule
- **Daily (12:00 UTC)**: Capture daily notes from memory/ directory
- **Weekly (13:00 UTC, Sunday)**: Summarize insights and patterns
- **Monthly (14:00 UTC, 1st)**: Review trends and update learnings

## Self-Improvement Framework
- **What broke / what went wrong**: Document the event
- **What I learned**: Extract the principle
- **What I'll do differently**: Create actionable rule

## N8n Integration
The vault is maintained through N8n workflows that ensure consistent, automated maintenance:
- Daily capture workflow
- Weekly summary workflow
- Monthly review workflow

## Purpose
This system transforms raw daily notes into structured knowledge that:
1. Preserves institutional memory
2. Identifies patterns and trends
3. Captures lessons learned
4. Enables continuous improvement
5. Supports better decision-making

## How to Use
1. **Capture**: Daily notes are automatically captured from memory/ directory
2. **Synthesize**: Weekly and monthly reviews extract insights and patterns
3. **Learn**: Lessons are documented in 05-Learnings/ folder
4. **Improve**: Recommendations are made based on observed patterns

## Notes
- This vault is maintained automatically through cron jobs
- Manual additions can be made to any folder as needed
- The system is designed to grow and improve over time
